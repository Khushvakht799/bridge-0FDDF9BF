# JVGr Specification v1.0

## Формат: JSON
## Расширение: .json или .jvgr

## Структура:
{
  "nodes": [
    {"id": "узел1", "type": "тип", "name": "имя"},
    {"id": "узел2"}
  ],
  "vectors": [
    {"from": "узел1", "to": "узел2", "strength": 0.85, "prob": 0.6}
  ]
}

## Поля:
- id: уникальный идентификатор (строка)
- from/to: ссылки на id узлов
- strength: сила связи (0.0-1.0, float)
- prob: вероятность (0.0-1.0, float)
- type, name, tags: опционально
